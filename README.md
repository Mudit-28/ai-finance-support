# 🧠 Finance AI
> Your AI-powered personal finance coach — understands not just what you spend, 
but why.

Most finance apps tell you where your money went. WealthMind tells you what your 
spending says about you. Built on **Groq-hosted Llama 3** for low-latency LLM 
inference, WealthMind combines transaction analysis with **behavioral AI** to 
identify spending personalities, emotional patterns, and habit loops — then 
coaches you toward better financial decisions in plain language.

## Key Features
- 🧠 Behavioral profiling — identifies spending personality and emotional triggers
- 💬 LLM-powered coach — natural language financial guidance and goal tracking
- 📊 Anomaly detection — flags unusual transactions and spending spikes
- 🔁 Subscription intelligence — detects and tracks recurring charges
- 📈 Proactive insights — weekly AI-generated financial health reports
- 🔐 Row-level security via Supabase — your data stays yours

## Tech Stack
| Layer | Technology |
|-------|-----------|
| LLM Inference | Groq API (Llama 3) |
| Behavioral Engine | Python |
| Database & Auth | Supabase (PostgreSQL + RLS) |
| Backend | Python |
